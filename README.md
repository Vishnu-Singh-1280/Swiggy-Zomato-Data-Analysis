# Swiggy-Zomato-Data-Analysis
📌 Project Overview
This project is an interactive Power BI dashboard built to analyze restaurant data from Swiggy and Zomato.
The goal of this analysis is to uncover insights related to:

Restaurant ratings
Pricing trends
Food categories
Location distribution

📊 Key Metrics
Total Restaurants: 115
Total Locations: 58
Average Cost for Two: ₹321

📈 Dashboard Features
🔹 KPI Cards
Total restaurants
Total locations

Average cost for two
🔹 Filters (Slicers)

Location filter
Rating filter
→ Enables dynamic and interactive analysis
🔹 Visualizations Used
Pie Chart:
Distribution of restaurants by rating
Bar Chart:
Comparison of restaurant count across rating categories
Donut Chart:
Relationship between cost and rating
Table:
Restaurant-level data (name + cost for two)
Column Chart:
Cost comparison across restaurants with ratings
Treemap:
Distribution of food categories (e.g., North Indian, Chinese, Biryani)
Map Visualization:
Geographic distribution of restaurants

🔍 Key Insights
Most restaurants fall in the 4.0–4.3 rating range
Higher ratings do not always mean higher prices
Mid-range pricing dominates the dataset
Popular food categories include:
North Indian
Chinese
Biryani
